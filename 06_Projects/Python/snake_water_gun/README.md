A simple Python command-line game based on the classic Snake, Water, Gun rules.

## Rules
- Snake drinks Water
- Water disables Gun
- Gun kills Snake

## How to Play
1. Run the program
2. Enter your choice:
   - s for Snake
   - w for Water
   - g for Gun
3. The computer randomly chooses
4. The winner is decided based on rules
